Решаем технологические задачи

Создаем для бизнеса индивидуальную технологическую стратегию, основанную на глубоком анализе отрасли и перспективных тенденциях. Внедряем эффективные решения для бесперебойной работы инфраструктуры, проводим компанию через все этапы цифровой трансформации. Так, для WEBBANKIR мы стали ключевым технологическим партнером, осуществив как большие, так и точечные задачи: запустили систему A/B тестирований, которая даст компании возможность проверять гипотезы клиентского опыта для увеличения эффективности бизнеса. Также разработали для них платформенное решение по управлению данными. Для СберРешений провели нагрузочные тестирования, сформировали рекомендации по оптимизации сервиса.

Создаем продукты для клиентов

Для крупнейшей розничной сети Узбекистана Korzinka мы разработали мобильное приложение онлайн-супермаркета. В первый год активными пользователями стали 130 тысяч клиентов сети. Для стартапа Veelzy разработали приложение для велосипедистов, которое объединяет в себе социальные и навигационные сервисы: пользователь может спланировать поездки с учетом погоды, получает бонусы за активность, может видеть передвижения друзей и переписываться с ними. Для рекламного агентства разработали AI платформу, позволяющую создавать сайты на основе сгенерированных AI текстов и изображений. Для экосистемы ПравоТех сделали систему автоматизации юридических процессов, которая помогает управлять делами, проектами, документами, задачами и отчетами в едином интерфейсе.

Разрабатываем финтех проекты

Финтех - одна из тех сфер, в которой мы обладаем максимальной компетенцией. Наши ключевые сотрудники много лет работали в компаниях-лидерах финансовой отрасли. В финтехе мы становимся партнером для банков, МФО и стартапов, связанных с финансами. Разработали сайт для нового банка Узбекистана с интеграцией внутренних систем, автоматизацией анкетного сервиса, внедрением онлайн-заявок. Организовали единое хранилище данных для группы компаний Мигкредит. Разработали платежный шлюз, позволяющий проводить оплату российских услуг и товаров картами, выпущенными за границей.

В последние годы активно развивается сфера BNPL (от англ. buy now, pay later — «покупай сейчас, плати потом»). Мы разработали c «нуля» BNPL платформу Frilly, которая позволяет клиенту оплачивать покупки частями, а бизнесу - увеличивать продажи и поднимать средний чек. Провели интеграцию с интернет магазинами, кредиторами и платежными компаниями. Сделали BNPL сервис для «Denum Плати частями». Для стартапа из Нидерландов разработали сервис платежей, позволяющий компаниям проводить платежи для своих клиентов в любой точке мира.

Мы также готовы реализовывать нестандартные задачи. Так, мы разработали «под ключ» платформу для проведения розыгрышей Национальной лотереи в одной из стран СНГ. Для кибербез-хаба в центре Москвы «Кибердом разработали сайт, приложение и личный кабинет для резидентов, реализовали систему управления пространством (умный дом).
